# Reports

## Overview

The Reports section provides comprehensive financial and operational analytics for your LEGO resale business. Access detailed insights on profitability, inventory health, platform performance, and purchase ROI through six specialized reports.

## Available Reports

| Report | Description | Key Metrics |
|--------|-------------|-------------|
| [Profit & Loss](./profit-loss.md) | Monthly income and expense tracking | Net Profit, Margin %, Category Breakdown |
| [Inventory Valuation](./inventory-valuation.md) | Current stock value assessment | Cost Value, Sale Value, Potential Profit |
| [Inventory Aging](./inventory-aging.md) | Stock age distribution analysis | Age Brackets, Slow-Moving Items |
| [Platform Performance](./platform-performance.md) | Sales comparison across platforms | Revenue, Fees, Best Platform |
| [Purchase Analysis](./purchase-analysis.md) | ROI tracking per purchase | ROI %, Mileage Costs, Source Performance |
| [Daily Activity](./daily-activity.md) | Daily listing and sales tracking | Items Listed, Sold, Store Status |

## Common Features

### Date Range Selection

Most reports support flexible date filtering:

| Preset | Description |
|--------|-------------|
| **This Month** | Current calendar month |
| **Last Month** | Previous calendar month |
| **This Quarter** | Current quarter (Jan-Mar, Apr-Jun, etc.) |
| **Last Quarter** | Previous quarter |
| **This Year** | January 1st to today |
| **Last Year** | Previous full calendar year |
| **Last 12 Months** | Rolling 12-month period |
| **Custom** | User-selected date range |

### Export Functionality

All reports support data export:

- **CSV** - Spreadsheet-compatible format
- **JSON** - Structured data for analysis

Click the **Export CSV** or **Export** button in the report header.

### Charts

Reports include interactive visualizations:

| Chart Type | Used In |
|------------|---------|
| **Bar Chart** | P&L trends, ROI by purchase, platform fees |
| **Pie Chart** | Value distribution, order distribution |
| **Combo Chart** | Profit vs revenue trends |

Charts support tooltips showing exact values on hover.

## Use Cases

### Monthly Business Review

1. Open **Profit & Loss** with "This Month" preset
2. Review income, expenses, and net profit
3. Check **Platform Performance** for channel analysis
4. Export data for accounting records

### Inventory Health Check

1. Open **Inventory Aging** report
2. Identify items in 91+ day brackets (slow-moving)
3. Check **Inventory Valuation** for total stock worth
4. Plan clearance sales for old stock

### Purchase Decision Making

1. Before buying, review **Purchase Analysis**
2. Check ROI by source (which buying channels perform best)
3. Consider mileage costs for collection purchases
4. Compare against target margin

### Daily Operations

1. Open **Daily Activity** report
2. Track listing progress against targets
3. Record store status (open/closed/holiday)
4. Compare performance across platforms
