# User Journey: eBay Listing Integration

> **Journey:** Create eBay listings directly from inventory items
> **Entry Point:** Inventory item row â†’ "List on eBay" action
> **Complexity:** High

## Overview

The eBay integration allows creating professional eBay listings directly from inventory items. The process includes photo upload, AI-generated descriptions, pricing configuration, and Best Offer settings - all in a multi-step modal interface.

## Prerequisites

- eBay account connected via OAuth (Settings â†’ Integrations)
- Business policies configured in eBay Seller Hub
- Inventory item not already listed (`ebay_listing_id` is null)

2. **Uploading photos** - Uploads to eBay picture service

4. **Creating eBay listing** - Calls eBay API to create listing
5. **Running quality review** - AI analyses listing quality
