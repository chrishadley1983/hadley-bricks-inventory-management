# User Journey: Amazon Arbitrage (BrickLink Sourcing)

> **Journey:** Compare Amazon selling prices against BrickLink sourcing prices
> **Entry Point:** `/arbitrage/amazon`
> **Complexity:** High

## Overview

The Amazon Arbitrage page helps identify profitable opportunities by comparing your Amazon selling prices against BrickLink "New" condition sourcing prices. Items with a positive margin (Amazon price > BrickLink price) represent potential arbitrage opportunities.

## Steps

### 1. View Summary Dashboard

**Action:** Navigate to `/arbitrage/amazon`

**What's Shown:**
- Summary cards with key metrics
- Sync status for all data sources
- Tabbed interface for opportunities/unmapped/settings

### 2. Trigger Data Sync

**Action:** Click "Full Sync" or individual sync buttons

**What Happens:**
1. API calls to Amazon SP-API for inventory and pricing
2. API calls to BrickLink for price guide data
3. Updates `arbitrage_sync_status` table
4. Refreshes UI with new data

**Sync Jobs:**
| Job | API Used | Data Fetched |
|-----|----------|--------------|
| Amazon Inventory | SP-API Listings | Your ASINs, quantities |
| Amazon Pricing | SP-API Pricing | Buy box, your price, offers |
| BrickLink Pricing | Price Guide API | Min/avg/max prices by condition |

### 3. Browse Opportunities

**Action:** View the opportunities table

**Table Columns:**
| Column | Description |
|--------|-------------|
| Item | Product image, name, BrickLink set number |
| Your Price | Your current Amazon listing price |
| Buy Box | Current buy box price (or lowest offer) |
| BL Min | Minimum BrickLink "New" price (UK) |
| Margin | Calculated profit margin percentage |
| Rank | Amazon sales rank |
| Action | Link to BrickLink |

**Row Highlighting:**
- Green background: Opportunity (margin â‰¥ threshold)
- White background: Below threshold

## Technical Details

### Query Keys

```typescript
arbitrageKeys.list(filters)     // List with filters
arbitrageKeys.item(asin)        // Single item
arbitrageKeys.excluded()        // Excluded ASINs
arbitrageKeys.unmapped()        // Unmapped ASINs
arbitrageKeys.syncStatus()      // Sync status
arbitrageKeys.summary(minMargin)// Summary stats
```

### Filter Parameters

```typescript
interface ArbitrageFilterOptions {
  minMargin?: number;           // Default: 30
  show?: 'all' | 'opportunities' | 'in_stock' | 'zero_qty' | 'pending_review';
  sortField?: 'margin' | 'bl_price' | 'sales_rank' | 'name';
  sortDirection?: 'asc' | 'desc';
  search?: string;
  page?: number;
  pageSize?: number;            // Default: 50
}
```

### Amazon Offer Structure

```typescript
interface AmazonOffer {
  sellerId: string;
  condition: string;            // "New", "Used", etc.
  subCondition: string;
  fulfillmentType: 'AFN' | 'MFN'; // FBA or Merchant Fulfilled
  listingPrice: number;
  shippingPrice: number;
  totalPrice: number;
  currency: string;
  isPrime: boolean;
}
```

### BrickLink Price Detail

```typescript
interface BricklinkPriceDetail {
  quantity: number;
  unit_price: string;
  seller_country_code: string;
}
```
